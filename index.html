01 Introduction to Python
DEFINITION — Python

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum in 1991. It emphasizes code readability and simplicity using English-like syntax.

Key Features

  • Interpreted — Code runs line by line; no compilation needed
  • Dynamically Typed — No need to declare variable types
  • High-Level — Closer to human language than machine language
  • Object-Oriented — Supports classes and objects
  • Open Source — Free to use and distribute
  • Cross-Platform — Runs on Windows, Mac, Linux
  • Indentation-based — Uses spaces/tabs instead of braces {}

Applications of Python

Web DevelopmentData Science Machine LearningAutomation Game DevelopmentCybersecurity ScriptingDesktop Apps

02 Variables & Identifiers
DEFINITION — Variable

A variable is a named memory location used to store data. In Python, variables are created when you assign a value. No declaration keyword is needed.

DEFINITION — Identifier

An identifier is the name given to a variable, function, class, or module. Rules: starts with letter/underscore, can contain letters/digits/underscores, case-sensitive, cannot be a keyword.

Rules for Variable Names

  • Must start with a letter (a–z, A–Z) or underscore _
  • Can contain letters, digits (0–9), and underscores
  • Cannot start with a digit — 2name is invalid
  • Case sensitive — name ≠ Name ≠ NAME
  • Cannot use Python keywords (if, for, while, etc.)

06 Conditional Statements
DEFINITION — Conditional Statement

Conditional statements allow a program to make decisions and execute different code blocks based on whether a condition is True or False.

Types

  • if — executes block if condition is True
  • elif — checks another condition if previous was False
  • else — executes if all conditions are False
  • Nested if — if inside another if
  • Ternary / Inline if — one-line condition

marks = int(input("Enter marks: ")) if marks >= 90: print("Grade: A+") elif marks >= 75: print("Grade: A") elif marks >= 60: print("Grade: B") elif marks >= 40: print("Grade: C — Pass") else: print("Grade: F — Fail")

08 Break, Continue & Pass
break

Immediately exits the loop when encountered. The remaining iterations are skipped.

continue

Skips the current iteration and moves to the next one. Loop does not stop.

pass

A null statement — does nothing. Used as a placeholder when a statement is syntactically required but no code is needed.

Python — break, continue, pass
# break — stop at 5 for i in range(1, 10): if i == 5: break print(i, end=" ") print("\n---") # continue — skip even numbers for i in range(1, 10): if i % 2 == 0: continue print(i, end=" ") print("\n---") # pass — empty block placeholder for i in range(3): pass print("pass loop done")

(no input)
◀ OUTPUT

---

09 Functions
DEFINITION — Function

A function is a reusable block of code that performs a specific task. Defined using the def keyword. Reduces repetition and improves code organization.

Types of Function Arguments

  • Positional — passed in order: add(2, 3)
  • Default — has default value: def greet(name="Guest")
  • Keyword — passed by name: add(b=3, a=2)
  • *args — variable number of positional args (tuple)
  • **kwargs — variable number of keyword args (dict)
Python — Functions (all types)
# Basic function def greet(name): print(f"Hello, {name}!") # Function with return value def add(a, b): return a + b # Default argument def power(base, exp=2): return base ** exp # *args def total(*nums): return sum(nums) # **kwargs def info(**details): for k, v in details.items(): print(f"{k}: {v}") # Calling functions greet("Alice") print(add(5, 3)) print(power(3)) print(power(2, 8)) print(total(1, 2, 3, 4, 5)) info(name="Bob", age=25, city="Delhi")

10 Lambda Functions
DEFINITION — Lambda

A lambda is an anonymous (nameless) single-line function. Written as: lambda arguments: expression. Used for short, throwaway functions.

17 Object-Oriented Programming (OOP)
DEFINITION — OOP

OOP is a programming paradigm that organizes code using classes and objects. It is based on 4 pillars: Encapsulation, Inheritance, Polymorphism, Abstraction.

TermDefinition
ClassBlueprint/template for creating objects
ObjectInstance of a class — has state (attributes) and behavior (methods)
__init__()Constructor — automatically called when object is created
selfReference to the current instance of the class
InheritanceChild class inherits properties from parent class
EncapsulationBundling data + methods; restricting direct access
PolymorphismSame method name behaves differently in different classes
Python — Class, Object, Inheritance
# Base class class Animal: def __init__(self, name, sound): self.name = name self.sound = sound def speak(self): print(f"{self.name} says {self.sound}") # Child class (Inheritance) class Dog(Animal): def __init__(self, name): super().__init__(name, "Woof") def fetch(self): print(f"{self.name} fetches the ball!") class Cat(Animal): def __init__(self, name): super().__init__(name, "Meow") # Create objects d = Dog("Rex") c = Cat("Whiskers") d.speak() c.speak() d.fetch() # Polymorphism for animal in [d, c]: animal.speak() # same method, different result

21 Comprehensions
DEFINITION — List Comprehension

A concise one-line syntax to create new lists from existing iterables. Syntax: [expression for item in iterable if condition]

Python — List, Dict, Set Comprehensions
# List comprehension squares = [x**2 for x in range(1, 6)] print(squares) # [1, 4, 9, 16, 25] # With condition evens = [x for x in range(1, 11) if x%2==0] print(evens) # [2, 4, 6, 8, 10] # Dict comprehension sq_dict = {x: x**2 for x in range(1, 5)} print(sq_dict) # {1:1, 2:4, 3:9, 4:16} # Set comprehension unique = {x%3 for x in range(10)} print(unique) # {0, 1, 2} # Generator expression gen = (x**2 for x in range(5)) print(list(gen)) # [0, 1, 4, 9, 16]
