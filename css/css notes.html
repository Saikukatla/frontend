Cascading Style Sheets — From Selectors to Layouts

01 What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation of HTML documents — colors, fonts, spacing, layout, animations, and more.


  • HTML → Structure & Content
  • CSS → Style & Presentation
  • JavaScript → Behavior & Interactivity
02 Ways to Add CSS

1. Inline CSS

Applied directly on the element using the style attribute. Highest specificity but hard to maintain.

<h1 style="color: red;">Hello</h1>

2. Internal CSS

Written inside a <style> tag in the <head>. Applies to one HTML page only.

<style>
p { color: blue; }
</style>

3. External CSS

Stored in a separate .css file and linked via <link>. Best practice for large projects.

<link rel="stylesheet" href="style.css">
03 CSS Selectors

Selectors define which HTML elements a CSS rule applies to.

Element
p { }

Targets all <p> tags

Class
.same { }

Targets class="same"

ID
#lite { }

Targets id="lite"

Universal
* { }

Targets ALL elements

Attribute
[type="text"]

Targets by attribute

Group
h1, h2 { }

Targets multiple

Descendant
div p { }

p inside any div

Child
div > p { }

Direct child only

Pseudo-class
a:hover { }

On mouse hover

Pseudo-element
p::first-line

Style first line

04 Specificity & Cascade

Specificity (Higher wins)

1
!important — Overrides everything
2
Inline style — style="..."
3
ID selector — #id
4
Class / Attribute / Pseudo-class — .class
5
Element selector — p, h1
6
Universal selector — *

From your code: *{color:black} is overridden by p{color:blue}, then .same{color:green}, then #lite{color:red}.

08 The CSS Box Model

Every HTML element is a box. From outside to inside:

margin
border
padding
content

12 Flexbox

Apply display: flex to a container to make children flex items.

.container {
display: flex;
flex-direction: row | column;
justify-content: flex-start | center | space-between | space-around;
align-items: stretch | center | flex-start | flex-end;
flex-wrap: wrap | nowrap;
gap: 16px;
}
.item {
flex: 1; /* grow equally */
align-self: center; /* override align-items */
}
13 CSS Grid
.grid {
display: grid;
grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
grid-template-columns: repeat(3, 1fr); /* same shorthand */
grid-template-rows: auto 200px auto;
gap: 20px;
}
.item {
grid-column: 1 / 3; /* spans columns 1 to 3 */
grid-row: 1 / 2;
}

15 Responsive Design & Media Queries
/* Mobile first approach */
.container { width: 100%; }

/* Tablet (768px and up) */
@media (min-width: 768px) {
.container { max-width: 720px; }
}

/* Desktop (1024px and up) */
@media (min-width: 1024px) {
.container { max-width: 960px; }
}

/* Max-width approach (tablet and below) */
@media (max-width: 600px) {
nav { display: none; }
}
16 Transitions & Animations

Transitions

button {
background: blue;
transition: background 0.3s ease; /* property duration timing */
}
button:hover { background: darkblue; }

Keyframe Animations

@keyframes fadeIn {
from { opacity: 0; }
to { opacity: 1; }
}

.box {
animation: fadeIn 1s ease-in-out forwards;
/* name duration timing fill-mode */
}
17 CSS Variables (Custom Properties)
/* Declare in :root (global) */
:root {
--primary-color: #7c3aed;
--font-size: 16px;
--spacing: 8px;
}

/* Use with var() */
button {
background: var(--primary-color);
font-size: var(--font-size);
padding: var(--spacing);
}
